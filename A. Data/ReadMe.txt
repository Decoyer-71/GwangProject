1. Raw data
- 전처리 전 상태인 데이터들을 공유하고 싶으시면 Raw Data 폴더에 넣어주세요
- 링크 폴더도 내부에 포함했습니다. 
	[광진통합지도서비스.xls]
		광진생활MAP > 광진통합지도서비스 에서 받은 공영주차장 엑셀파일

2. Preprocessing Data
- Raw data 중 전처리 중이거나 전처리 하고 있는 데이터들은 Preprocessing Data 폴더에 
자신의 이름 폴더를 만들고 넣어주세요

	ex) Preprocessing Data 폴더 - 윤진석 폴더 - 광진구 건축물 면적 데이터_전처리 중
